# 🌎⚽ Figuritas Mundial 2026

Proyecto de colección digital Mundial 2026 con funcionalidad de descarga optimizada.

## [🌐 Clic para Ver o descargar álbum online](https://figuritas-mundial-2026-gratis.vercel.app/)

---

## 🏆 Mundial 2026

Próxima Copa del Mundo organizada en:

- 🇺🇸 Estados Unidos
- 🇲🇽 México
- 🇨🇦 Canadá

---

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript vanilla
- **Backend**: Vercel Serverless Functions (Node.js)
- **Compresión**: Archiver.js (streaming ZIP)
- **Hosting**: Vercel

---

## ⚡ Optimizaciones

- **Streaming ZIP**: Los archivos ZIP se generan en tiempo real sin ocupar espacio en disco
- **Cache inteligente**: Headers de cache para reducir tiempos de carga
- **Lazy loading**: Imágenes cargadas bajo demanda
- **Compresión balanceada**: Nivel 6 para país, nivel 5 para álbum completo
